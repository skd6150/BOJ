dp[y][k][case] -> y높이에서 k개의 방을 닫은 case

case 0: y높이 두칸 다 열림 (1, 1)
case 1: y높이 왼쪽 닫힘 (0, 1)
case 2: y높이 오른쪽 닫힘 (1, 0)

점화식)
y높이에서 k개의 방을닫은 0번 케이스)
	y-1높이에서 k개의 방을 닫은 0번케이스 + y높이의 양쪽 방 value
	y-1높이에서 k개의 방을 닫은 1번케이스 + y높이의 양쪽 방 value
	y-1높이에서 k개의 방을 닫은 2번케이스 + y높이의 양쪽 방 value
	3개 중 가장 큰 값
	
y높이에서 k개의 방을닫은 1번 케이스)
	y-1높이에서 k-1개의 방을 닫은 0번 케이스 + y높이의 오른쪽 방 value
	y-1높이에서 k-1개의 방을 닫은 1번 케이스 + y높이의 오른쪽 방 value
	2개 중 가장 큰 값
	
y높이에서 k개의 방을닫은 2번 케이스)
	y-1높이에서 k-1개의 방을 닫은 0번 케이스 + y높이의 왼쪽 방 value
	y-1높이에서 k-1개의 방을 닫은 2번 케이스 + y높이의 왼쪽 방 value
	2개 중 가장 큰 값
