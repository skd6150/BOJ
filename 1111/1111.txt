연립방정식 계산

1. n < 3 일때
    n == 1 일때 'A'
    n == 2 이고 arr[0] != arr[1] 일때 'A'
    n == 2 이고 arr[0] == arr[1] 일때 arr[0]

2. n >= 3 일때
    a, b를 연립방정식으로 구할 수 있음.
    arr[0] == arr[1] 일 때, division by zero 발생, a=0, b=arr[0]으로 고정

    a, b를 int로 선언해서 a, b가 실수일 경우 정확하지 못한 값이 저장되서 이후 조건에서 탈락

    모든 arr 원소에 대해 a, b를 검증 하면서 값이 다르면 'B'
    모든 값이 규칙에 맞으면 arr[n-1] * a + b
